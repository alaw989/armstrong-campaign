# Campaign Team Photos

Place team member headshots in this directory.

## Requirements
- Square or 4:5 aspect ratio (400x400 or 400x500px)
- Professional headshot style
- JPG or PNG format
- Naming: {name}.jpg (e.g., jane-smith.jpg)

## Alt Text
Update components/MembersSection.vue with proper alt text: "{Name}, {Role}"
