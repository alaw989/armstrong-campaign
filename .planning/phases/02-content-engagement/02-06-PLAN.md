# Design Reference
Follow https://diijondacosta.com/ contact pattern:
- Three-column layout (Email | Phone | Social)
- Dark background (gray-900) for visual separation
- Icon circles with brand color (teal-600)
- Social media links as circular buttons
- Volunteer CTA centered below columns

# Phase 1 Patterns
- Use Composition API with <script setup lang="ts">
- Semantic HTML (section, a, footer, nav)
- Tailwind utility classes for styling
- Section IDs for Navigation scroll spy
- Use app.config.ts for configurable content
