# Design Reference
Follow https://diijondacosta.com/ hero pattern EXACTLY:
- Full-width hero with min-h-screen
- Split layout on desktop: content left (md:col-span-1), photo right (md:col-span-1)
- On mobile: photo first (order-1), content second (order-2)

Hero content structure (left column on desktop):
1. "Re-Elect" badge (if applicable) or geographic badge:
   - Small, uppercase, tracking-widest
   - text-teal-800 font-semibold text-xs tracking-wider uppercase
   - Above candidate name

2. Candidate name (h1):
   - text-4xl md:text-6xl font-bold text-gray-900
   - Leading-tight for tight line spacing

3. Position title:
   - text-2xl md:text-3xl text-gray-700 font-medium
   - Below name, no extra spacing

4. Tagline/slogan:
   - text-xl text-gray-600
   - Separate paragraph with mt-4

5. CTA buttons:
   - Grouped in flex flex-col sm:flex-row gap-4
   - Donate (primary variant, large size)
   - Volunteer (secondary/outline variant, same size)

Candidate photo (right column on desktop):
- Aspect ratio: 3:4 (portrait) or 1:1 (square)
- Use NuxtImg with format="webp" for optimization
- Classes: rounded-2xl shadow-2xl
- Decorative background: gradient or shape behind photo
- On mobile: max-w-xs mx-auto (centered, not full width)
</context>
