# Design Reference
Follow https://diijondacosta.com/ patterns:
- About: Clean typography with section headers (Who I Am, Why I'm Running, Vision)
- Platform/Leadership: Numbered cards (01, 02, 03...) with hover effects
- Card grid: Responsive (1 col mobile, 2 col tablet, 3 col desktop)
- Section backgrounds alternating between white and light gray/teal

# Phase 1 Patterns
- Use Composition API with <script setup lang="ts">
- Semantic HTML (section, article, h2, h3)
- Tailwind utility classes for styling
- Section IDs for Navigation scroll spy
</context>
