# Design Reference
Follow https://diijondacosta.com/ patterns:
- Members: Round headshots, name, role, grid layout
- Endorsements: Blockquote-style cards with quote, name, title
- Contact/Get Involved: Three-column layout (Volunteer | Donate | Stay Informed)
- Social links: Icon buttons in footer and contact section

# Phase 1 Patterns
- Use Composition API with <script setup lang="ts">
- Semantic HTML (section, article, blockquote, cite, footer)
- Tailwind utility classes for styling
- Section IDs for Navigation scroll spy
- Use app.config.ts for configurable content
</context>
