# Design Reference
Follow https://diijondacosta.com/ CTA pattern EXACTLY:

Hero CTA placement (from 02-01):
- Large button (size="large")
- Primary variant (teal-800 background)
- Positioned at bottom of hero content left column
- Full width on mobile, auto on desktop

Navigation CTA placement:
- Desktop: After nav links, before mobile hamburger button
- Size="small" or size="xs" for compact fit
- Same primary variant as hero for consistency
- Hidden on mobile (hidden md:block)

Mobile nav CTA:
- Inside mobile menu panel
- Full width button (class="w-full")
- Size="medium" for touch targets
- Positioned at bottom of mobile menu (pt-2 spacing)

After-platform CTA placement:
- Centered below card grid (class="text-center mt-12")
- Secondary variant (yellow-500 background) for visual distinction
- Large size for prominence
- Contextual text above: "Want to support Xzandria's platform for Houston County?"
- Text: "Donate to the Campaign" (more specific than just "Donate")

# FEC Compliance Requirements (LEGAL-02, LEGAL-03)
- No pre-checked recurring donation boxes
- Clear indication of one-time vs recurring options
- Transparent fees and easy cancellation information
- Link to external platform (ActBlue/Anedot) handles FEC compliance
