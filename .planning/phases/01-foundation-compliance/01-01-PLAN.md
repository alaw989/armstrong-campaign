  <name>Initialize Nuxt 4 project with SSG configuration</name>
  <files>
    nuxt.config.ts
    package.json
    .gitignore
    app.config.ts
  </files>
  <action>
    Create a new Nuxt 4 project from scratch in the current directory:

    1. Initialize with: `npx nuxi@latest init . --package-manager npm --gitInit false`
    2. When prompted for project name, use "campaign-site"
    3. Accept defaults for TypeScript/ESLint/Prettier prompts

    Then create nuxt.config.ts with:
    - `ssr: true` (server-side rendering for static generation)
    - `nitro.preset: 'static'` (static site generation)
    - `modules: ['@nuxt/ui', '@nuxtjs/seo']` (install these modules)
    - `site` configuration with:
      - `url: 'https://armstrongforhouston.com'` (placeholder domain - update when actual domain known)
      - `name: 'Xzandria Armstrong for Houston County'`
      - `description: 'Leadership for Houston County, GA. Learn about Xzandria Armstrong's platform, vision, and campaign.'`
      - `defaultLocale: 'en'`
      - `indexable: true`
    - `sitemap` config: `enabled: true, autoLastmod: true, sortEntries: true`
    - `robots` config: `disallow: [], sitemap: 'https://armstrongforhouston.com/sitemap.xml'`
    - TypeScript enabled with `typescript: { strict: true, typeCheck: true }`

    Install additional dependencies: `npm install @nuxt/ui @nuxtjs/seo`

    Update .gitignore to include:
    - .output/
    - .nuxt/
    - node_modules/
    - dist/
    - .env

    Create app.config.ts for app-wide configuration (brand colors placeholder):
    ```ts
    export default defineAppConfig({
      site: {
        name: 'Xzandria Armstrong for Houston County',
        description: 'Leadership for Houston County, GA'
      }
    })
    ```

    IMPORTANT: Use Nuxt 4 (latest). Do NOT use Nuxt 2 or 3.
  </action>
  <verify>
    1. Run `npm run dev` - should start dev server on http://localhost:3000
    2. Run `npm run generate` - should produce .output/public/ directory with index.html
    3. Check that nuxt.config.ts contains `nitro.preset: 'static'`
    4. Check that package.json includes @nuxt/ui and @nuxtjs/seo in dependencies
  </verify>
  <done>
    Nuxt 4 project initialized with:
    - Static site generation configured (nitro.preset: 'static')
    - TypeScript enabled in strict mode
    - @nuxt/ui and @nuxtjs/seo modules installed
    - Site metadata configured for SEO
    - `npm run generate` produces static .output/public/index.html
  </done>
